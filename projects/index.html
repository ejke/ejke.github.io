What I did was that I put the whole quiz togerther again, using HTML5 this time - all animations and special effects are made using CSS3. iOS/Android app

Concepthome app

Concepthome was our client who wanted their customers to be able to browse and save house plans in their phone. This is exactly what this app allows you to do - to see house plans and save the ones you like most for reviewing them later. My task was to take mock-ups from designer and turn them into prototype. I used Framework7 to make hybrid app for both iOS and Android. More thoughts on it can be read here.

On another app project I used Ionic2 for hybrid app, which was so much easier to implement and more powerful than Framework7. That second project has not been released yet, therefore I cannot hand out any link for futher inspectation.
